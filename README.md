# DooSan-Robotics-TTL



### ttb_gui
---

터틀봇 인터페이스

주문 정보를 받아서 네비게이션에게 전달하는 패키지 입니다. 

```bash
ros2 run ttb_gui delivery_order.py 
```

토픽 pub 

```bash
 ros2 run  ttb_gui delivery_topic.py 
```

로봇 액션 서버
```bash
 ros2 run ttb_gui delivery_server.py 
```
